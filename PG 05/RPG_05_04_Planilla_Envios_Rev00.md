# RPG 05 04 - Planilla de Envíos a Clientes

**Revisión:** 01  
**Estado:** Documento Operativo / Registro Logístico

---

## Descripción del Registro
Este documento registra la salida de materiales, equipos reparados o repuestos enviados a clientes, asegurando la trazabilidad de la entrega.

## Campos Mínimos Requeridos

| Campo | Descripción |
| :--- | :--- |
| **Fecha de Envío** | Fecha de despacho. |
| **Cliente** | Razón social o nombre del destinatario. |
| **Dirección de Envío** | Lugar físico de entrega. |
| **Transportista** | Empresa de logística / Correo / Comisionista. |
| **N° de Guía/Seguimiento** | Tracking ID para trazabilidad. |
| **Contenido del Paquete** | Detalle de equipos o repuestos enviados. |
| **N° de Ticket/Caso** | Referencia al caso de soporte asociado. |
| **Estado** | En tránsito / Entregado / Devuelto. |

---

> **Nota del SGC:** La preservación de los comprobantes de envío (remitos, guías) forma parte integral de este registro. En el nuevo esquema de AppSheet, se recomienda adjuntar la foto del comprobante al registro digital.
