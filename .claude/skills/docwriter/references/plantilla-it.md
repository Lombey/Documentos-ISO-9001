## Notas

- IT no tiene secciones fijas como PG/PP (OBJETIVO, ALCANCE, etc.)
- Estructura libre según el contenido técnico
- Header usa "Sistemas:" en vez de "Fecha Revisión"
- Ubicación típica: `/ISO9001/PG 05/Manuales Técnicos/`
- Ajustar path del CSS según ubicación (`../../../` si está en subcarpeta)
