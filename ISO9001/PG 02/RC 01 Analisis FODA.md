### Metodología de Evaluación

**Probabilidad (P):** 1 (Muy Baja) — 5 (Muy Alta)

**Impacto (I):** 1 (Insignificante) — 5 (Catastrófico)

**Nivel de Riesgo (P×I):**

| Rango | Nivel | Criterio |
| :---: | :--- | :--- |
| 1–4 | Bajo | Aceptable — monitoreo pasivo |
| 5–14 | Medio | Requiere acción con plazo y responsable |
| 15–25 | Alto | Acción inmediata |

**Estados posibles:**

| Estado | Significado |
| :--- | :--- |
| Vigente | Activo, con seguimiento en curso |
| Fuera de alcance | Excluido del SGC, registrado para trazabilidad |
| Cerrado | Riesgo resuelto o desaparecido |

**Eficacia de las acciones** — evaluada en planificación (ene/feb) y Revisión por la Dirección (jul/sep):

| Valor | Significado |
| :--- | :--- |
| Eficaz | La acción funcionó en el período evaluado |
| Ineficaz | La acción no produjo el resultado esperado |
| Parcialmente eficaz | Resultado mixto o incompleto |
| Sin datos | Acción nueva o no iniciada, sin evidencia aún |
| — | No aplica (Fuera de alcance / sin acción asignada) |

*Para riesgos Vigentes la evaluación es un snapshot a la fecha de revisión — no implica cierre del riesgo. Para riesgos Cerrados es definitiva.*
