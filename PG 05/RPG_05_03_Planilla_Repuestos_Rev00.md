# RPG 05 03 - Planilla de Repuestos

**Revisión:** 00  
**Estado:** Documento Operativo / Registro de AppSheet (Pendiente de Validación)

---

## Descripción del Registro
Este documento constituye la evidencia objetiva del control de inventario y movimiento de repuestos utilizados en el servicio de soporte técnico y post venta.

## Campos Mínimos Requeridos

| Campo | Descripción |
| :--- | :--- |
| **Fecha** | Fecha del movimiento. |
| **Tipo de Movimiento** | Entrada / Salida / Devolución / RMA. |
| **Código de Pieza** | Identificador único del repuesto. |
| **Descripción** | Detalle del componente. |
| **Cantidad** | Unidades movidas. |
| **Destino/Origen** | Cliente, Técnico, Proveedor o Depósito. |
| **N° Ticket/Orden** | Vinculación con el caso de soporte (RPG 05 01). |
| **Responsable** | Persona que registra el movimiento. |

---

> **Nota del SGC:** Este registro puede estar digitalizado e integrado en la plataforma AppSheet o sistema de gestión de stock vigente. La evidencia se valida mediante la trazabilidad del N° de Ticket.
