# PP 05 06 - Procedimientos para nuevos equipos provenientes de concesionarios

**Revisión:** 00  
**Fecha:** 29/09/2025

---

## 1. OBJETIVO
Definir y establecer la metodología para asegurar el funcionamiento y configuración del equipo y la debida instrucción del cliente para equipos nuevos provenientes por ventas de concesionarios.

## 2. ALCANCE
Servicio de Post Venta.

## 3. DESARROLLO DEL PROCESO
El proceso comienza cuando un cliente se comunica con el soporte de Corvus y el agente de soporte identifica que el cliente adquirió un equipo por venta a través de “concesionarios”.

### Pasos Generales
1.  Se realizan las verificaciones y configuraciones en el sistema.
2.  Se instruye al cliente.
3.  Se actualizan los registros.

### 4. VERIFICACIONES Y CONFIGURACIÓN EN SISTEMA

#### 4.1 Realizar asignación a concesionario
1.  Ingresar a **Maruiz**.
2.  Ir a pantalla **“Balanzas”**.
3.  En el B-T-H- a asignar, seleccionar lápiz de edición.
4.  Seleccionar el concesionario que corresponda y grabar.

#### 4.2 Crear nueva tolva para el B-T-H- en concesionario
1.  Ingresar al usuario del concesionario.
2.  Ingresar al menú de tolva > Seleccionar **Nueva Tolva**.
3.  Ingresar número de serie y modelo de la tolva (datos obtenidos de planillas compartidas).
4.  Asignar el B-T-H- y guardar.

#### 4.3 Generar usuario contratista
1.  Ingresar a **Maruiz** > Pantalla **“Contratistas”** > Nuevo.
2.  Ingresar los datos de usuario y grabar.
3.  Compartir con el cliente el acceso generado y el link `www.agpd.com.ar`.

#### 4.4 Asignar la tolva al contratista
1.  Ingresar a **Maruiz** > Pantalla **“Tolvas”**.
2.  Buscar el B-T-H- y editar.
3.  Seleccionar el usuario contratista a asignar y guardar.

### 5. INSTRUCCIÓN DEL CLIENTE
El agente de soporte transmitirá al cliente los aspectos básicos de la utilización del sistema (instalación corriente, diferencias con balanza tradicional, uso de la tablet, carga de establecimientos en web, etc.). Se debe informar sobre el período de prueba y posterior licencia.

### 6. ACTUALIZACIÓN DE REGISTROS EN BASE DE DATOS
*   **Solapa TABLET:** Actualizar USUARIO, CONTRASEÑA, BALANZA, VENDEDOR, etc.
*   **Solapa CONECTIVIDADES:** Actualizar USUARIO, DATOS CONTACTO, MES VENCIMIENTO. En observaciones indicar el concesionario del cual procede.
